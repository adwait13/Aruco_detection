1. Create a folder to save image files, in the code it's named 'droidcam_images'
2. Take a charuco board. Click 15-20 pictures from different angles using 'getImages.py' script
3. Then run 'calib_charuco.py'
4. You should get these files generated: cameraMatrix.pkl, caliberation.pkl, dist.pkl
5. Run 'main.py' for detecting aruco markers, their pos etc.
